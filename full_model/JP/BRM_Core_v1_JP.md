# BRMコア原則 v1.0

## 定義

BRM（Behavioral Reference
Memory）とは、AIとの相互作用における連続性を制御するモデルであり、文脈の安定性を確率的補完ではなく、明示的な構造参照によって維持するものである。

## 基本原則

### 1. 構造的連続性原則

連続性はテキストの蓄積ではなく、明示的な構造参照によって維持される。

### 2. 明示的アンカー形成原則

アンカーは明示的な確定プロセスを通じてのみ生成される。

### 3. 非生成アンカー原則

生成出力は既定状態においてアンカーとなってはならない。

### 4. 参照基盤ジャンプ解決原則

思考の不連続は確率的補完ではなく、既存アンカーの参照によって解決されなければならない。

### 5. セッション独立性原則

連続性は会話セッションに依存しない。

## 核心的洞察

BRMは記憶を「会話文脈の蓄積」から「明示的に確定された構造的連続性」へと再定義する。
